# Interactive Photo Gallery
Project working link: https://jellu28.github.io/Slider-Gallery-1/

![Project Screenshot](img/screenshot.png)

A sleek, interactive image gallery featuring smooth horizontal scrolling, hover effects, and custom navigation controls.

## 🧠 Features
- **Horizontal Scroll**: Navigate through images using the mouse wheel or trackpad.
- **Smooth Navigation**: Custom "Back" and "Next" buttons with smooth scroll behavior.
- **Visual Effects**: Grayscale images that transition to full color and scale up on hover.
- **Modern Layout**: Built using CSS Grid and Flexbox for a clean, structured look.

## 📱 Compatibility Note
This project is currently **optimized for Desktop screens (1440px)**. 
Mobile responsiveness is limited, and the layout is best experienced on larger displays. 


## 🛠️ Technologies Used
- **HTML5**
- **CSS3** (Custom properties, Grid, Flexbox)
- **Vanilla JavaScript** (DOM manipulation & Event Listeners)
- **FontAwesome**: Used for navigation icons (loaded via CDN).

## 🗝 How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Use the side arrows or your mouse wheel to browse the collection.
